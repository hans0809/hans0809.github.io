开篇

所谓拆包，就是将一个可迭代对象中的元素拆解成多个变量。

以列表这个可迭代对象为例来说，我们将多个元素用列表存储是为了减少变量的个数，而拆包的目的正好相反，它是将一个列表中的多个元素分别用一个单独的变量来表示，从而变量的个数会增加。

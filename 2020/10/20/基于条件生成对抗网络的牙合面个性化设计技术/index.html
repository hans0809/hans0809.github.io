基于条件生成对抗网络的牙合面个性化设计技术
发表于 2020-10-20| 更新于 2020-12-23|论文阅读
阅读量:

摘要

This article proposes an intelligent network model for designing tooth crown surface based on conditional generative adversarial networks. Then, the data set for training the network model is constructed via generating depth maps of 3D tooth models scanned by the intraoral. Through adversarial training, the network model is able to generate tooth occlusal surface under the constraint of the space occlusal relationship, the perceptual loss, and occlusal groove filter loss. Finally, we carry out the assessment experiments for the quality of the occlusal surface and the occlusal relationship with the opposing tooth. The experimental results demonstrate that our method can automatically reconstruct the personalized anatomical features on occlusal surface and shorten the treatment time while restoring the full functionality of the defective tooth.

文章提出了一种基于条件生成对抗网络的用于设计牙冠表面的智能网络模型。然后，通过生成由口腔内扫描的3D牙齿模型的深度图，构造用于训练网络模型的数据集。通过对抗训练，该网络模型能够在空间咬合关系，感知损失和咬合槽过滤器损失的约束下生成牙齿咬合面。最后，我们进行了咬合面质量以及与对牙的咬合关系的评估实验。实验结果表明，我们的方法可以自动重建咬合面上的个性化解剖特征，并缩短治疗时间，同时又恢复了矫正牙齿的全部功能。

1.介绍

Via the data-driven intelligent design mode, this article provides a new methodology for the tooth occlusal surface reconstruction based on conditional generative adversarial networks (CGANs). The main contributions of this study include following.

  • First, we propose a novel method that using the combination of the CGANs network and the perceptual loss to reconstruct the defect tooth.
  • Second, in order to fulfill functional requirements of repaired crowns, we use the opposing tooth information with space occlusal relationship to restrict the output of network model.
  • Third, we take a pretrained encoding-decoding generator as the occlusal groove filter network, and the filter network is fixed and added into the reconstruction network to control the occlusal surface morphology.
  • Finally, the depth image of the reconstructed crown is converted to a 3D model for the restoration fabrication.

通过数据驱动的智能设计模式，本文提供了一种基于条件生成对抗网络（CGAN）的牙齿咬合面重建的新方法。这项研究的主要贡献包括以下内容:

  • 首先，我们提出了一种新颖的方法，该方法结合了CGANs网络和感知损失来重建缺损牙齿。
  • 其次，为了满足修复后的牙冠的功能要求，我们使用了具有空间咬合关系的对牙信息限制了网络模型的输出。
  • 第三，以训练好的编解码生成器作为咬合槽滤波器网络，将滤波器网络固定并添加到重建网络中，以控制咬合面的形态。
  • 最后，将重建冠的深度图像转换为用于修复纤维的3D模型 。

2.方法

2.1 技术步骤

Three steps ：

  • Building data sets: The training samples of the reconstruction network and occlusal groove filter network are 2D images. Therefore, we set a suitable plane to generate depth images for the tooth preparation, the opposing tooth,the occlusal groove extracted by technicians, and the restoration designed by technicians.
  • Applying different loss functions: In this article, we use three loss functions to constraint the output of the reconstruction network.
    • First, we use the L1 loss function to offer a constraint for deviation between the generated crown surface and the technicians’ design.
    • Second, we combine the adversarial loss with the perceptual loss to penalize the high-level features discrepancy between both crown surfaces.
    • Finally, we add a fixed occlusal groove filter network to filter occlusal grooves and make the occlusal groove on generated crown closer to the one of technician’s design crowns.
  • Experiment evaluation: Several criteria are chosen to evaluate the error between the depth image of generated crown surface and the depth image of technicians’ design. After that, we transform the 2D depth image of the crown to a 3D triangular mesh model. Afterward, there are several criteria of the quality assessment for 3D models being used to measure the deviation for occlusal surface morphology, which consist of the curvature similarity, the absolute deviation, and the interference distance with the opposing tooth

共3个步骤：

  • 建立数据集：重建网络和咬合沟过滤器网络的训练样本为2D图像。因此，我们设置了一个合适的平面来生成深度图，用于牙齿准备，对牙，技术人员提取的咬合沟以及技术人员设计的修复体。

  • 应用不同的损失函数：在本文中，我们使用三个损失函数来约束重构网络的输出。

    • 首先，我们使用L1损失函数作为生成的胎冠表面和技术人员的设计之间的偏差提供约束。
    • 其次，我们将对抗损失与感知损失相结合，以惩罚两个冠面之间的高级特征差异。
    • 最后，我们增加了一个固定的咬合槽过滤网络，以过滤咬合槽，并使生成的牙冠上的咬合槽更接近技术人员的设计牙冠之一。

  • 实验评估：选择几个标准来评估生成的冠面深度图像和技术人员设计的深度图像之间的误差。之后，我们将冠的2D深度图像转换为3D三角形网格模型。之后，有3D模型的质量评估标准用于测量咬合面形态的偏差，包括曲率相似性，绝对倾斜偏差以及与之相对的干涉距离。

2.2 生成深度图

In this article, we construct the depth maps (256×256） that contain information relating to the distance of the tooth surface to a given plane from some viewpoint. Then, we build a reference plane with a suitable distance from the tooth model in the direction, which is parallel to the occlusion plane of the dental jaw and divided into 256×256 grids.

在本文中，我们构造了深度图（256×256），该深度图包含从某些角度来看与牙齿表面到给定平面的距离有关的信息。然后，我们在一个方向上建立一个与牙齿模型相距适当距离的参考平面，该参考平面与牙颌的咬合平面平行并分成256×256网格。

3. 实验设置

3.1 数据集

根据相关研究报道，下颌第一磨牙缺损率最高，我们选择36颗牙齿缺失的患者作为修复对象。
在训练咬合沟滤波网络的过程中，数据集包含500个样本。
每个样本都包含咬合槽和相应的原始牙齿(36号磨牙与相邻牙齿)的深度图。
对于重建网络，数据集包含500个用于训练的样本和100个用于测试的样本。
每个样本分为三类，包括准备工作的深度图、对侧牙齿的深度图和技师设计的牙冠(以及相应的邻牙)。
牙冠在咬合运动过程中经过反复测试和调整，冠面包含动态咬合关系记录。
实验牙齿除缺损外无其他疾病，咬合分类中的对侧牙齿为正常咬合，该方法也适用于其他牙齿。

3.2 网络结构和训练细节

本文中的咬合槽滤波器网络和咬合面重建网络与pix2pix类似。 G的编码器架构为“ C64-C128-C256-C512-C512-C512-C512-C512”，并具有激活功能隐藏层数是LReLU。G的解码器架构为“ CD512-CD512-CD512-CD512-C256-C128-C64”，并使用ReLU作为激活函数。D的编码器架构为“ C64-C128-C256-C512”，其激活功能与模型G的编码器架构相同。具体而言，后者将感知损失和咬合关系与对置牙齿结合在一起。隐藏层“ C128-C256-C512”用于测量生成的表冠和技术人员设计之间的感知差异。用于训练咬合沟过滤器网络的超参数值与pix2pix相同，表1列出了在训练重建网络期间的超参数值。

为了有效地训练重建网络，我们采用了课程学习策略。整个训练过程包括三个阶段。首先，我们使用相应的训练数据集训练咬合沟过滤器网络并冻结模型。其次，我们将训练后的模型中的发生器作为咬合槽的滤失，此外，利用滤失和L1损失指导模型G生成具有合理的咬合面形态和咬合槽类型的牙冠。最后，使用感知损失来微调网络的输出以生成许多解剖特征。

4.讨论与分析

4.1 生成咬合表面质量评估

目前在图像质量和相似性评估研究中采用了几种基本方法，例如峰值信噪比（PSNR），多尺度结构相似性指数（MS-SSIM），均方根误差（RMSE）和特征相似度索引（FSIM）。此外，PSNR和RMSE已成为评估压缩和修复图像质量的常用方法。

通常，针对目标图像，两个条件都为处理后的图像提供基于像素的误差。与PSNR和RMSE不同，MS-SSIM和FSIM分别是基于结构的度量和基于特征的度量，这与主观评估更加一致。基于上述四个指标，我们对测试数据集进行质量评估，并对评估结果取平均值。结果的平均值示于表3，评估结果的散点图示于图4。

4.2 咬合关系评估

当我们重建检测牙齿的形态时，实现正常的咬合关系非常重要。为了恢复缺损牙齿的全部功能，上下后牙应保持微干涉。此外，根据牙医的建议，干扰距离应在0.02至0.05 mm的范围内。在本文中，我们测量了干扰距离并以频率直方图的形式给出了距离。如图6所示，生成的由相对牙齿限制的咬合面（图6C）在咬合间距离的分布上与技术人员的设计最相似。图7表明，在没有对置齿约束的情况下，对牙冠的干扰过大，并且在产生有对置齿约束的牙冠时，可以满足重建牙冠表面的原理。
